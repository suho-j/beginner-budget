# 요구사항

## 목표

처음 가계부를 쓰는 사용자가 복잡한 설정 없이 이번 달 돈 흐름을 이해하고 거래를 기록할 수 있게 한다.

## 기능 요구사항

- 첫 화면에서 이번 달 또는 선택한 달의 수입, 지출, 잔액, 예산 남은 금액을 표시한다.
- 거래 입력 필드는 날짜, 유형, 카테고리, 금액, 메모로 구성한다.
- 유형은 `수입`과 `지출`을 지원한다.
- 지출 카테고리: 생활비, 배달비, 의류비, 비상금
- 수입 카테고리: 월급, 용돈, 부수입, 기타
- 월 예산을 설정할 수 있으며 기본값은 500,000원이다.
- 월별 필터와 유형 필터를 제공한다.
- 삭제는 `confirm` 확인 후 처리한다.
- 샘플 데이터 채우기와 전체 초기화를 제공한다.
- 데이터는 `localStorage` 키 `beginner-budget-app:v1`에 저장한다.

## 비기능 요구사항

- 빌드 도구 없이 HTML/CSS/JavaScript만 사용한다.
- 앱 파일은 `beginner-budget-app/` 내부에만 둔다.
- UI와 문서는 한국어로 작성한다.
- 잘못된 JSON 또는 빈 저장소에서도 앱이 멈추지 않아야 한다.
- 금액은 양의 정수만 허용한다.
- 모바일 우선으로 구현하고 360px 폭에서 가로 스크롤이 없어야 한다.
- 모든 `input`과 `select`는 `label`과 연결한다.
- 색상만으로 의미를 전달하지 않고 텍스트도 함께 제공한다.
