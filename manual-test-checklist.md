# 수동 테스트 체크리스트

## 시작

- [ ] `index.html`을 브라우저에서 열 수 있다.
- [ ] `python3 -m http.server`로 실행한 주소에서도 열 수 있다.
- [ ] 첫 화면에 수입, 지출, 잔액, 예산 남은 금액 카드가 보인다.
- [ ] 기본 예산이 500,000원으로 표시된다.

## 입력

- [ ] 오늘 날짜가 기본으로 입력된다.
- [ ] 유형을 수입으로 바꾸면 수입 카테고리가 표시된다.
- [ ] 유형을 지출로 바꾸면 지출 카테고리가 표시된다.
- [ ] 양의 정수 금액으로 거래를 추가할 수 있다.
- [ ] 금액이 비어 있거나 0 이하이면 오류 메시지가 나온다.
- [ ] 메모 없이도 거래를 추가할 수 있다.

## 요약과 목록

- [ ] 지출 추가 후 지출 합계와 예산 남은 금액이 바뀐다.
- [ ] 수입 추가 후 수입 합계와 잔액이 바뀐다.
- [ ] 목록에 날짜, 유형, 카테고리, 금액, 메모가 표시된다.
- [ ] 새로고침 후에도 데이터가 유지된다.

## 필터

- [ ] 월별 필터가 해당 월 거래만 보여준다.
- [ ] 유형 필터 `전체`가 모든 거래를 보여준다.
- [ ] 유형 필터 `수입만`이 수입만 보여준다.
- [ ] 유형 필터 `지출만`이 지출만 보여준다.

## 삭제/데이터 도구

- [ ] 삭제 버튼을 누르면 확인 창이 뜬다.
- [ ] 삭제 확인 후 목록과 요약이 갱신된다.
- [ ] 샘플 데이터 채우기로 샘플 내역이 추가된다.
- [ ] 전체 초기화 전 확인 창이 뜬다.
- [ ] 전체 초기화 후 기본 예산과 빈 목록으로 돌아간다.

## 접근성/반응형

- [ ] 모든 입력과 선택 상자에 라벨이 연결되어 있다.
- [ ] 360px 폭에서 가로 스크롤이 없다.
- [ ] 수입/지출/예산 상태가 색상뿐 아니라 텍스트로도 구분된다.
